The registration form includes a built-in theme switcher that allows users to toggle between light and dark modes. This guide shows you how the theme system works and how to customize the color schemes.

## Dark Mode Toggle

The form includes a theme toggle button that switches between light and dark modes:

```html
<button id="change-darkAndLigth">Cambiar color</button>
```

The toggle is implemented with a simple event listener that adds/removes the `dark-mode` class:

```javascript
document.getElementById("change-darkAndLigth").addEventListener("click", () => {
    document.body.classList.toggle("dark-mode")
})
```

## Light Theme (Default)

The default light theme uses warm, paper-like colors to create a sticky note aesthetic:

### Body Background

```css
body {
    background:
        radial-gradient(circle at 20% 30%, rgba(139, 90, 43, 0.1) 0%, transparent 50%),
        radial-gradient(circle at 80% 70%, rgba(160, 102, 51, 0.1) 0%, transparent 50%),
        linear-gradient(135deg, #c19a6b 0%, #d4a574 50%, #c19a6b 100%);
}
```

### Form Boxes

The form uses two distinct box colors:

**Box 1** (Yellow sticky note):
```css
.box-1 {
    background: linear-gradient(135deg, #fffacd 0%, #fef9c0 100%);
}
```

**Box 2** (Green sticky note):
```css
.box-2 {
    background: linear-gradient(135deg, #d4f4dd 0%, #c8f0d0 100%);
}
```

## Dark Mode Overrides

When the `dark-mode` class is applied to the body, the following overrides are activated:

### Body Background

```css
body.dark-mode {
    background: #1a1a1a !important;
    color: #ffff !important;
}
```

### Form Boxes

```css
body.dark-mode .box-1,
body.dark-mode .box-2 {
    background: #2a2a2a !important;
    color: #ffff !important;
    border-color: #444 !important;
}
```

### Input Fields

```css
body.dark-mode input,
body.dark-mode textarea,
body.dark-mode select {
    background: #333 !important;
    color: #fff !important;
    border-color: #555 !important;
}
```

### Headings

```css
body.dark-mode h1,
body.dark-mode h2,
body.dark-mode h3 {
    color: #ffff !important;
}
```

### Buttons

```css
body.dark-mode button {
    filter: brightness(0.8);
}
```

## Customizing Colors

<Steps>
  <Step title="Choose your color scheme">
    Decide on your primary colors for light and dark modes.
  </Step>
  
  <Step title="Update light theme colors">
    Modify the gradient values in the `body`, `.box-1`, and `.box-2` selectors.
  </Step>
  
  <Step title="Update dark mode colors">
    Modify the `body.dark-mode` and related selectors to match your dark theme.
  </Step>
  
  <Step title="Test the toggle">
    Click the "Cambiar color" button to test both themes.
  </Step>
</Steps>

### Example: Blue Theme

Here's how to create a blue-themed light mode:

```css
body {
    background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
}

.box-1 {
    background: linear-gradient(135deg, #e0f2fe 0%, #bae6fd 100%);
}

.box-2 {
    background: linear-gradient(135deg, #dbeafe 0%, #bfdbfe 100%);
}
```

## Customizing the Toggle Button

The theme toggle button has custom styling:

```css
#change-darkAndLigth {
    background: linear-gradient(135deg, #f56565 0%, #e53e3e 100%);
    padding-left: 10px;
    border-radius: 0px 10px 10px 0px;
    margin-top: 40px;
}

#change-darkAndLigth:hover {
    background: linear-gradient(135deg, #dd4545 0%, #e53e3e 100%);
    transition: 4s;
}
```

<Note>
You can change the button text in `index.html` from "Cambiar color" to any text you prefer, such as "Toggle Theme" or "Dark/Light Mode".
</Note>

## Advanced: Gradient Customization

The body background uses multiple gradient layers for depth. You can customize each layer:

```css
body {
    background:
        /* Radial gradient at top-left */
        radial-gradient(circle at 20% 30%, rgba(139, 90, 43, 0.1) 0%, transparent 50%),
        /* Radial gradient at bottom-right */
        radial-gradient(circle at 80% 70%, rgba(160, 102, 51, 0.1) 0%, transparent 50%),
        /* Linear gradient base */
        linear-gradient(135deg, #c19a6b 0%, #d4a574 50%, #c19a6b 100%);
}
```

Modify the percentages (20%, 30%, 80%, 70%) to reposition the radial gradients, or change the RGBA values to adjust colors and opacity.

<Tip>
When customizing themes, maintain sufficient contrast between background and text colors for accessibility. Test both modes thoroughly.
</Tip>
