## Overview

Club Las Aves is a modern, interactive sports club registration form designed for ease of use and accessibility. Built with vanilla JavaScript, HTML, and CSS, it provides a seamless user experience with real-time form validation, dynamic sport selection, and theme customization.

## Key Features

- **Smart Validation**: Real-time validation for name, age, email, phone number, and sport selection
- **Visual Feedback**: Instant color-coded indicators (green for valid, red for errors) with descriptive messages
- **Spanish Phone Validation**: Specialized validation for Spanish mobile numbers (starting with 6, 7, or 9)
- **Dynamic Sports List**: Add up to 5 additional sports beyond the default 4 options
- **Dark Mode**: Toggle between light and dark themes for comfortable viewing
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Animated UI**: Smooth animations and transitions for a polished user experience
- **Local Storage**: Automatic form data persistence across sessions

## Technology Stack

The form is built with modern web technologies:

- **HTML5**: Semantic markup with accessibility features
- **CSS3**: Modern styling with gradients, animations, and responsive design
- **Vanilla JavaScript**: No frameworks required - pure ES6+ JavaScript
- **Animate.css**: For smooth, professional animations
- **Local Storage API**: For data persistence
